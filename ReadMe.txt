TRIDENTS by Emmet H

First build, just for testing, very incomplete

Underwater: Up to swim up, left/right to move
Above water: Left/right to rotate, up to throw
Both: Down to recall tridents

Player 1 (yellow) : Arrow Keys
Player 2 (green)  : WASD
Player 3 (magenta): IJKL
Player 3 (gray)   : 8456 (Numpad)

'n' to delete gray
'm' to delete magenta
'b' to toggle middle block
